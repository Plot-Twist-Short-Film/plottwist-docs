Plot Twist Set Project

Updates current DCC Plot Twist project path to point to proper folder.

Plot Twist Set Project

Why is important?

When working inside DCCs, they work taking into account the current Project Path, this is the path that will be used as the root path when working with relative paths.

To properly work in Plot Twist you need to setup the path where Artella downloads Plot Twist file locally. You can manually setup the path but with this tool the path will be setup automatically.

How to use it

Just left-click in top of the icon in Plot Twist Shelf.

Important

Plot Twist Launcher automatically after launching the DCC, so you do not need to do it if you launch DCC using Plot Twist Launcher.
