Plot Twist Assets Manager

This is the main tool to manage the different assets available for Plot Twist project.

Using this tool you can:

  • Open Asset Files (model, shading, rig, etc)
  • Lock & Unlock Asset Files
  • Synchronize Asset Files

Kitsu

Kitsu is the Production Manager tool that we are using in Plot Twist.

Important

Plot Twist Assets Manager intensively uses the information stored in Plot Twist Kitsu database to show different information of the assets. For this reason, is mandatory to be connected to the Internet when the tool is launched.

Login into Kitsu

Plot Twist Assets Manager - Login into Kitsu

If you launch Plot Twist Assets Manager and no Kitsu login information is detected you will have to login into Kitsu.

Kitsu Login Toggle button - Not Logged

Note

The first time you launch Plot Twist Assets Manager you will need to login into Kitsu.

Plot Twist Assets Manager - Kitsu Login Dialog

To login into Kitsu, you need to left-click on top ot the Kitsu Login Toggle button. This will show Kitsu Login Dialog in the center of your screen.

Kitsu Login Dialog has tree fields you can edit:

  • Email: In this field you need to type the email you used during Kitsu registration.
  • Password: In this field you need to type the password you used during Kitsu registration.
  • Store Credentials: If you check this checkbox, your credentials will be store and you won't have to type them again.

Once you are ready, click on Login button to continue.

If your login credentials are valid, Plot Twist assets info will be loaded.

Assets Viewer

The Assets Viewer is where all Plot Twist Assets are loaded.

Asset

Asset

An Assets Viewer Asset is composed by:

  • Image that shows how the asset is visually.
  • Name that shows the name of the asset.
Contextual Menu

Asset Contextual Menu

If you right-click on top of one of the assets, the contextual menu of the asset will appear, showing the following options:

Name Description
Open in Kitsu Open Asset Kitsu webpage in your default web browser.
Open in Artella Open Asset Artella webpage in your default web browser.
View Locally Open local folder where Asset files are located in your computer.
Synchronize Open asset synchronization options.

Asset Synchronization Options

Here you can find the different types of asset synchronization currently supported:

File Type Description
All Synchronizes all the files of the asset.
Textures Synchronizes texture files of the asset.
Shading Synchronizes shading files of the asset.
Model Synchronizes model files of the asset.
Rig Synchronizes rig files of the asset.
Groom Synchronizes groom files of the asset.

Note

Take into account that depending of the type asset some files are not available. For example, grooming files are only available for characters.
